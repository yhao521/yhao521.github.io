三、放开端口

  • 打开特定端口： iptables -A INPUT -p tcp --dport <端口号> -j ACCEPT
  • 保存规则： /etc/rc.d/init.d/iptables save
  • 重启服务： /etc/init.d/iptables restart
  • 查看端口是否开放： iptables -L -n
