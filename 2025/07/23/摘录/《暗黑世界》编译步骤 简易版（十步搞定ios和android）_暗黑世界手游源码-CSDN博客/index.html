本文详细介绍了使用cocos2d-x2.1.4版本进行游戏开发的步骤，包括下载、配置环境、导入源码、编译运行等操作，并提供了Android版本的导入和编译指南。

摘要生成于 C知道 ，由 DeepSeek-R1 满血版支持， 前往体验 >

欢迎来到9秒：www.9miao.com

开发环境：Mac + Xcode + Eclipse + Android SDK + NDK r8b

新手请仔细阅读步骤，碰到问题可以留言。
第一步：到cocos2d-x官网下载cocos2d-x 2.1.4 版本。
第二步：到置顶帖下载《暗黑世界》源码。
第三步：到coco2d-x 2.1.4目录下面建立project目录，将暗黑世界源码解压到其中（正确目录应该是： cocos2d-x 2.1.4\project\DiabloWorld）
第四步：打开暗黑世界目录中proj.ios中的tsDemo.xcodeproj工程。
第五步：选择工程和tsDemo的target 为5.1，选择好对应的模拟器，编译运行。即可进入游戏。
下面是android步骤：
第六步：打开配置好android的eclipse。选择workspace为project目录。
第七步：点击eclipse File菜单，选择Import从现有工程导入。选择根目录为暗黑世界目录。选择工程为proj.android工程。
第八步：打开Terminal， cd 到proj.android，执行chmod +x build_native.sh加上执行权限。
第九步：删除proj.android目录下面的lib和obj目录。到Terminal下面执行./build_native.sh。
第十步：待编译完成之后在eclipse下面点击clean，然后点击运行即可进入游戏。

几点说明：
1.不需要添加cocos2d-x到系统路径中， 保证目录正确直接执行build_native.sh 即可。
2.第九步要删除libs和obj目录，因为jie同学将目标文件也打包放到源码中了。否则会提示路径错误。
3.点击登录游戏如果长时间没有反映，可能是大鸡蛋没有开启服务器。明天发放服务器代码再说。
4.eclipse中只要导入proj.android即可，不用导入cocos2d-x的android工程。
5.如果没有Mac，按相应的步骤可以在windows下面生成Android版本。 有同学说生成不了，请参考其他帖子。
6.奉劝一句新手不要折腾vs工程了，大鸡蛋没打算支持。别浪费时间。
