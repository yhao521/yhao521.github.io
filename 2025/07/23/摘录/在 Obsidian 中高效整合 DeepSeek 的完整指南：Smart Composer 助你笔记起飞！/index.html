在 Obsidian 中高效整合 DeepSeek 的完整指南：Smart Composer 助你笔记起飞！

在 Obsidian 中高效整合 DeepSeek 的完整指南：Smart Composer 助你笔记起飞！

大家好，我是小明同学。

想让你的 Obsidian 笔记拥有更强大的 AI 创作能力？ Smart Composer 插件，结合 DeepSeek 大模型，将为你带来前所未有的写作体验！ 本文将手把手教你如何在 Obsidian 中配置和使用 Smart Composer，让 DeepSeek 成为你的专属写作助手。

项目地址： github.com/glowingjade/

![[_resources/d6afd05bf932ca6eefc11e90b0f12cc4_MD5.jpg]]

什么是 Smart Composer？

Smart Composer 是一款专为 Obsidian 用户设计的插件，灵感来自于 Cursor 的 Composer 。这款插件的目标是通过 AI 的力量，帮助用户快速生成文档，优化写作过程。与传统的 AI 工具不同，Smart Composer 允许你直接引用 Obsidian Vault 中的内容，让 AI 在你已有的知识基础上进行创作和编辑，从而确保内容的精准度和一致性。

为什么选择 DeepSeek + Smart Composer？

  • DeepSeek 的强大能力： DeepSeek 作为国内领先的大模型，在中文理解和生成方面表现出色，尤其擅长代码生成和逻辑推理。
  • Smart Composer 的无缝集成： Smart Composer 插件将 DeepSeek 的能力无缝集成到 Obsidian 中，让你在笔记中轻松使用 AI 功能。
  • 隐私安全： DeepSeek 提供 API 接口，你可以选择将数据发送到 DeepSeek 服务器，也可以选择使用本地部署的 DeepSeek 模型，更好地保护你的隐私。
  • 高效便捷： Smart Composer 提供了多种快捷操作，让你在写作过程中随时调用 AI 功能，提高写作效率。

准备工作

1.安装 Obsidian： 确保你已经安装了 Obsidian 笔记软件。

![[_resources/302ebb3d3aed4bff4612697c8a455e00_MD5.jpg]]

Obsidian下载地址： obsidian.md/download

2.安装 Smart Composer 插件：

  • 打开 Obsidian，进入 “设置” -> “第三方插件”->”社区插件市场”。
  • 禁用 “安全模式”。
  • 点击 “浏览”，搜索 “Smart Composer” 并安装。
  • 安装完成后，启用 Smart Composer 插件。

![[_resources/65efbaf666c28453ca2f0263e9049636_MD5.jpg]]

![[_resources/a9c88781e35f2ef2bc8b6b85f3666d25_MD5.jpg]]

![[_resources/9b141d49bc3f958899f35825e457fc09_MD5.jpg]]

3.获取 DeepSeek API Key：

  • 访问 DeepSeek 官网： deepseek.com/
  • 注册并登录 DeepSeek 账号。
  • 在 “API Keys” 页面创建并获取你的 API Key。

我这里演示使用的是硅基流动， 注册即送 2000 万 Tokens ，其他模型设置同理。 地址： 硅基流动 访问 API 密钥 新建密钥，名字自定义。

![[_resources/ba491db7bf580bad6538e02ef79a99b6_MD5.jpg]]

Smart Composer 常用功能

1.上下文智能助手 (Contextual Chat) ：

Smart Composer 的最大亮点之一是它的 上下文智能助手 。与其他传统的 AI 插件不同，Smart Composer 允许你精确选择对话的上下文。你只需在对话框中输入 @<文件名> ，就可以选择 Vault 中的特定文件或文件夹作为对话背景，从而让 AI 在更准确的上下文中生成回应。

  • 选择文件/文件夹 ：可以直接选择笔记中的文件或文件夹作为背景。
  • 快速理解需求 ：AI 基于选定的内容生成答案，而无需你反复提供背景信息。
  • 提高输出精准度 ：这种精确的上下文选择让内容创作更具针对性和深度。

![[_resources/c052bce01a1c155502eb1eb710511599_MD5.jpg]]

2.多媒体上下文支持 (Multimedia Context)：

  • Smart Composer 还支持多种类型的上下文，包括网站内容、图片和视频链接等，帮助你在写作时引入丰富的信息源。
  • 网站链接 ：可以直接添加网站链接，Smart Composer 会自动提取网页内容作为参考。
  • 图片支持 ：支持上传图片、拖放或从剪贴板粘贴图片。
  • YouTube 支持 ：YouTube 视频的文字稿将自动提取，并作为上下文参考。
  • 未来更新 ：即将支持 PDF、DOCX 等外部文件，进一步拓展插件的应用场景。

![[_resources/341414225fb15c76f5ce97c5b58864e9_MD5.jpg]]

3.文档编辑建议 (Apply Edit)

在创作过程中，Smart Composer 会根据现有的文档内容提供优化建议。通过这种智能化的推荐，创作者可以在不离开插件的情况下快速改进文档内容。

  • 智能文档改进 ：根据 AI 的分析，插件会推荐改进建议，帮助你提高文档质量。
  • 一键应用 ：你可以直接应用这些建议，节省手动编辑的时间。

![[_resources/453f70beed377b662677c18f6fd27fdc_MD5.jpg]]

4.Vault 搜索 (RAG)

Vault 搜索 是 Smart Composer 的另一大亮点。它通过语义搜索自动从你的 Vault 中寻找相关的笔记内容，帮助你快速找到与当前写作任务相关的信息。

  • 快捷搜索 ：只需按 Cmd+Shift+Enter ，便可启动 Vault 搜索。
  • 语义理解 ：AI 会根据语义理解来选择最相关的笔记内容作为上下文。

![[_resources/735a87c02ebeff3d8bea030b9a186c38_MD5.jpg]]

这一功能极大地提高了写作时的效率，让你不再需要手动查找和翻阅笔记。

5.模型上下文协议（MCP） 将Smart Composer连接到外部 MCP 服务器。MCP 允许您在聊天中使用强大的第三方工具和数据源。

![[_resources/56f8326ad02c12cb58009155080cecda_MD5.jpg]]

高级技巧

  • 使用本地 DeepSeek 模型： 如果你希望更好地保护你的隐私，或者想要在离线状态下使用 DeepSeek，你可以选择使用本地部署的 DeepSeek 模型。 具体步骤请参考 DeepSeek 官方文档。
  • 结合其他插件： Smart Composer 可以与其他 Obsidian 插件结合使用，例如 Templater、Dataview 等，实现更强大的功能。
  • Prompt Engineering ： 学习 Prompt Engineering 技巧，可以帮助你更好地控制 DeepSeek 的生成结果，获得更符合你需求的内容。
