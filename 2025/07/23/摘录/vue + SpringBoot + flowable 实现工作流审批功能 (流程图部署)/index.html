前端基础配置 bpmn 文件使用

参数(常用) 数据类型 含义
:xml String 流程图展示xml字符串
@save function 模型保存方法
:users [{name:’’,id:’’}] 用户信息
:groups [{name:’’,id:’’}] 组信息
:categorys [{name:’’,id:’’}] 流程分类
:is-view true|false 是否可编辑
